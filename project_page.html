Food Security and Crop Prices

This project helps shed light on the relationship between food prices and food security indicators. A correlation analysis revealed a positive and significant relationship between the three selected crop prices and the three selected indicators. A simple linear regression was run to get the coefficients and intercept for the three crops on each indicator. Each regression equation was then used in the chart “Predicted Food Security Indicators” to show the predicted food security indicator as the crop prices change. This analysis does not assign any causality to the relationship but does find that, in general, as crop prices increase, so does the level of food insecurity in the world.

The main data sources used are FAOSTAT (for historical prices and food security indicators) and Yahoo Finance (for crop prices).

Project Write Up

1. Project Aims

The aim of this project is to look at the relationship, if any, between crop prices and food security. The question originated from wondering about the impact on increasing food prices and if it does cause basic food needs to go unmet. This question is not meant to look for causal relationships, but rather aims to understand the presence of any relationship and then use that statistical relationship to predict the impact of crop prices on food security indicators.

2. The Data You Used and How It Was Accessed

The data for this project came from two main sources. The data for the food security indicators was downloaded directly from the FAOSTAT website. The FAO publishes a wide range of data related to food and agriculture for more than the past 10 years. The crop price data was scraped from Yahoo Finance. This data was scraped and aggregated at the year level in order to merge the crop prices with the data downloaded from the FAOSTAT site.

The level of data aggregation available for the food security indicators was a limiting factor in this project since it is only available on a yearly basis. The project was additionally limited since the crop prices available for scraping are not location specific. One way to improve this project and get a more comprehensive answer to the research question would be to pull all data at a monthly interval and pull crop price data for specific locations. Despite the data limitations, the available data does help address the research questions and provide more insight into the relationship between crop prices and food security.

3. Challenges in Data Cleaning and/or Analysis, and the Tools You Used to Overcome Them

Some of the main challenges in this project were deciding how and what data to pull. Several sites for pulling live commodity data required a paid API. Once the sources were identified, the cleaning did not take a significant amount of time or effort. The data from the FAO is relatively straightforward and only required some deleting of unnecessary data (whole categories also downloaded that were not part of the selected data) and converting values with the “<” symbol to a numeric value. I used Excel to manually clean the data when necessary.

The other tools and methods used in the project are tools and methods with which I am already familiar (mapping, regression, python), so the difficulties had more to do with data-specific issues and syntax errors.

4. Conclusions

This project helps shed light on the relationship between food prices and food security indicators. A correlation analysis revealed a positive and significant relationship between the three selected crop prices and the three selected indicators. A simple linear regression was run to get the coefficients and intercept for the three crops on each indicator. Each regression equation was then used in the chart “Predicted Food Security Indicators” to show the predicted food security indicator as the crop prices change. This analysis does not assign any causality to the relationship but does find that, in general, as crop prices increase, so does the level of food insecurity in the world.

Future analysis could be carried out to investigate this question at a more granular level either through a specific location, a more frequent interval, or both. The relationship between crop prices and food security is likely heavily dependent on location, geopolitical events, and other factors impacting purchasing power.
